# TypeScript Practice Problems Index

This directory contains a structured collection of TypeScript exercises designed to build your skills from basics to advanced patterns.

| Section | Topic | Key Features |
| :--- | :--- | :--- |
| **[01-Generics & Classes](./01-generics-and-classes/index.ts)** | Type Safety | Function generics, Generic Classes (`Stack`), and Tuple returns. |
| **[02-Object Manipulation](./02-object-manipulation/index.ts)** | Mapped & Utility Types | `keyof`, `Partial<T>`, and Intersection Types (`&`). |
| **[03-Type Guards & Narrowing](./03-type-guards-narrowing/index.ts)** | Runtime Safety | Custom type guards (`user is Admin`), `in` operator, and Narrowing. |
| **[04-Enums & Tuples](./04-enums-and-tuples/index.ts)** | Constants & Fixed Arrays | `enum` roles and fixed-length Tuples. |
| **[05-Advanced Conditional Types](./05-advanced-conditional-types/index.ts)** | Type Logic | `infer` keyword, Conditional Types (`T extends U ? X : Y`), and `ReturnType/Parameters`. |
| **[06-Async Fetching](./06-async-fetching/index.ts)** | Async Patterns | Generic `fetch` wrappers and `Promise<T>` handling. |

---

### How to use this directory:
1.  **Read the comments** at the top of each file to understand the topic.
2.  **Examine the implementations** to see the patterns in action.
3.  **Run the code** using `npx ts-node path/to/index.ts` to see the outputs in your terminal.
